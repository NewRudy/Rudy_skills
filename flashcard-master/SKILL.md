# Personal-Flashcard-Master

Automates the creation of high-quality flashcards for learning and memorization.

## Usage
- "Generate 5 Anki flashcards from this technical document."
- "Turn this article into a set of study questions."
- "Review these notes and identify the key concepts I need to memorize."

## Instructions
1. Extract atomic concepts from the provided text.
2. Format output as Markdown tables or CSV suitable for Anki import.
3. Follow the "Minimum Information Principle" for cards (one question per card).
