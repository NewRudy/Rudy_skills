# Life-Admin-Janitor

A global utility for cleaning up digital clutter and organizing local administrative files.

## Usage
- "Organize my Downloads folder."
- "Find all health-related documents and index them."
- "Rename and file these receipts into a 'Finance' folder."

## Instructions
1. Use file content analysis (not just extensions) to identify document types (e.g., invoices, medical reports, ID cards).
2. Propose a logical directory structure before moving any files.
3. Always ask for confirmation before bulk renaming or moving sensitive personal data.
