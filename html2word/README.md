# html2word - Word文档格式化生成技能

## 🎯 功能简介

通过 HTML 中间格式生成完美保留格式的 Word 文档，解决 AI 生成内容复制到 Word 后格式混乱的问题。

**核心优势：**
- ✅ 字体、字号、缩进完全保留
- ✅ 支持 LaTeX 数学公式（配合 VBA 宏自动转换）
- ✅ 多级列表、表格、图片完美呈现
- ✅ 一键复制粘贴，无需手动调整格式

## 📖 使用流程

### 1. 调用技能生成 HTML
```
/html2word 生成一份简历，包含个人信息、教育背景、工作经历
```

Claude 会生成一个 `.html` 文件（带完整的内联样式）

### 2. 浏览器预览
双击 `.html` 文件用浏览器打开，检查格式是否正确

### 3. 复制到 Word
- 浏览器中 `Ctrl+A`（全选）→ `Ctrl+C`（复制）
- Word 中 `Ctrl+V`（粘贴）
- **重要**：选择"保留源格式"粘贴

### 4. 转换数学公式（如果有）
如果文档包含数学公式：
1. Word 中按 `Alt+F11` 打开 VBA 编辑器
2. 点击"插入" → "模块"
3. 复制 HTML 注释中的 VBA 代码（或从 `vba_macro.vba` 文件复制）
4. 按 `F5` 运行宏
5. 所有 `$公式$` 自动转换为 Word 数学公式

- **行距**：1.5倍

## 🧮 数学公式支持

### 行内公式（inline）
使用单个美元符号 `$...$`：
```
质能方程 $E=mc^2$ 揭示了质量与能量的关系。
```

### 独立公式（display）
使用双美元符号 `$$...$$`：
```
二次方程的求根公式：
$$x = \frac{-b ± \sqrt{b^2-4ac}}{2a}$$
```

### 希腊字母
**推荐直接使用 Unicode 字符**（无需转换）：
- π（圆周率）
- α, β, γ, δ（常用小写）
- Δ, Σ, Ω（常用大写）

**或使用 LaTeX 命令**（VBA 宏会自动替换）：
- `\pi` → π
- `\alpha` → α
- `\sigma` → σ
- `\infty` → ∞
- `\nabla` → ∇

VBA 宏支持 **100+ LaTeX 符号**自动转换！

## 🔧 技术细节

### 为什么用 HTML 而不是 Markdown？
| 方法 | 格式保留 | 公式支持 | 易用性 |
|------|---------|---------|--------|
| Markdown → Word | ⚠️ 部分丢失 | ❌ 不支持 | ✅ 简单 |
| HTML → Word | ✅ 完美保留 | ✅ 完美支持 | ✅ 一键复制 |

### 关键技术点
1. **内联 CSS**：所有样式写在 `style=""` 属性中（Word 不识别 `<style>` 标签）
2. **pt 单位**：字体大小必须用 `pt`（磅），不能用 `px`（像素）
3. **表格居中**：必须用 `align="center"` 属性（CSS `margin: auto` 无效）
4. **UTF-8 编码**：确保中文正常显示
5. **VBA 预处理**：公式转换前替换 LaTeX 命令为 Unicode 字符

## ⚠️ 常见问题

### Q1: 粘贴到 Word 后格式丢失？
**A:** 确保选择"保留源格式"粘贴选项（左上角粘贴按钮旁的下拉菜单）

### Q2: 公式无法转换？
**A:** 检查：
1. 公式是否用 `$...$` 或 `$$...$$` 包裹？
2. VBA 宏代码是否完整复制？
3. 是否按 F5 运行了宏？

### Q3: 中文显示乱码？
**A:** 确保 HTML 文件包含 `<meta charset="UTF-8">`（生成时已自动添加）

### Q4: 表格不居中？
**A:** 检查 `<table>` 标签是否有 `align="center"` 属性（不是 CSS）

### Q5: 如何设置页眉页脚？
**A:** HTML 无法直接控制页眉页脚，需在 Word 中手动设置：
- 插入 → 页眉/页脚 → 选择样式
